# RegexLogin
 Login webpage, with regex verification

 Email:

 - Must have 2 characters before @
 - Must have at least 2 characters after @
 - Must have at least 2 characters after the . 

 - Exemple: xx@xx.xx

  Password:

 - Must Have at least:
    1 lowercase character
    1 uppercase character
    8 characters
    1 number character
    1 symbol character
